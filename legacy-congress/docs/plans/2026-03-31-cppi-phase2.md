```python
# cppi enrich --members          → cmd_enrich_members()

# cppi analyze weights           → cmd_analyze_weights()
# cppi diagnose member <id>      → cmd_diagnose_member()
# cppi backtest --start --end    → cmd_backtest()
```

New modules are imported lazily in command functions to avoid import-time dependencies on optional packages (Tesseract, yfinance).

## Graceful Degradation

| Dependency | Failure Mode | Behavior |
|------------|--------------|----------|
| Congress.gov API | Rate limited / down | Cache results, log warning, continue with cached data |
| Tesseract | Not installed | `cppi parse` skips paper filings with warning, logs count |
| Quiver API | No API key | `cppi validate --source quiver` fails with clear message about API key |
| Capitol Trades | Blocked/changed | `cppi validate --source capitol` fails gracefully, suggests Quiver |
| yfinance | Rate limited | Backtest uses cached prices, warns about staleness |

All external dependencies are **optional** — core CPPI functionality (ingest, parse, score, report) continues to work without them.
